# ChromeForensics-Research

ChromeForensics-Research is a DFIR-focused project for analyzing Chrome and Chromium browser artifacts.

The project explores browser forensic evidence such as browsing history, cookies, bookmarks, visited links, preferences, and SQLite databases to support digital investigations.

## Features

* Browser History Analysis
* Cookie Analysis
* Bookmark Extraction
* Visited Links Parsing
* SQLite Database Inspection
* Browser Metadata Analysis

## Technologies

* Java
* Gradle
* SQLite
* Swing GUI

## Future Enhancements

* PDF Report Generation
* Timeline Analysis
* Browser Extension Analysis
* IOC Detection
* Multi-Browser Support

## Status

**Work in Progress** — Currently being studied, documented, and enhanced for DFIR and cybersecurity research.
